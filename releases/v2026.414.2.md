# v2026.414.2

> Released: 2026-04-14

## Highlights

- **Desktop downloads are dramatically smaller on macOS** — The desktop release packager now builds compressed DMGs instead of near-uncompressed disk images, which cuts the macOS desktop download size from roughly `850 MB` to about `255–260 MB` while keeping the same packaged runtime contents. ([#60](https://github.com/penclipai/paperclip-cn/pull/60), @penclipai)

## Improvements

- **Release packaging now prefers compressed desktop artifacts across platforms** — Electron desktop packaging switched back from `store` compression to a release-oriented compressed profile, which also trims the Windows installer and Linux AppImage outputs compared with the previous desktop release settings. ([#60](https://github.com/penclipai/paperclip-cn/pull/60), @penclipai)
- **macOS desktop uploads are portable in GitHub Actions again** — The reusable desktop release workflow no longer relies on `mapfile`, so macOS runners can upload their desktop assets to the GitHub Release without the shell compatibility failure that forced manual recovery in the previous release. ([#60](https://github.com/penclipai/paperclip-cn/pull/60), @penclipai)

## Fixes

- **The desktop release workflow no longer produces oversized DMGs by default** — DMG output is now explicitly built as a compressed disk image, preventing future stable releases from regressing to the oversized uncompressed installer shape. ([#60](https://github.com/penclipai/paperclip-cn/pull/60), @penclipai)

## Upgrade Guide

- Existing installs can upgrade in place; this release changes desktop artifact packaging only.
- Desktop assets remain unsigned in this phase, and macOS continues to ship separate `x64` and `arm64` installers instead of a universal app.
