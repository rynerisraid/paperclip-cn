# v2026.508.2

> Released: 2026-05-08

## Highlights

- **Windows onboarding recovery** - `penclip onboard --yes` now recovers from interrupted embedded PostgreSQL startup attempts that leave stale embedded worker processes behind.

## Fixes

- **Embedded PostgreSQL startup** - Paperclip now removes only orphaned `@embedded-postgres` forkchild worker processes before starting the embedded database, avoiding the `running bootstrap script` init failure without touching unrelated PostgreSQL processes. ([#83](https://github.com/penclipai/paperclip-cn/pull/83), @penclipai)

## Upgrade Guide

- Upgrade to `2026.508.2` and rerun `npx penclip@latest onboard --yes`. If your local `npx penclip` still resolves an older cached package, use the explicit `@latest` form.
